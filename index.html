<a>

Anchor elemnt creates a hyperlink to other web pages, files, locations within the same page, email addresses, or any other URL.

This element's attributes include:

  • download
  • href
  • hreflang
  • ping
  • referrerpolicy
  • rel
  • target
  • type

Example:
<a href="https://example.com">Website</a>
<a href="#imprint">Back to top</a>
<br>

The Line Break element produces a line break in text (carriage-return). The text after the line break begins again at the start of the next line of the text block.

This element has only one attribute:

  • clear

Example:
Text line one <br>
Text line two
<div>

The Content Division element is the generic container for flow content. It has no effect on the content or layout until styled using CSS.

This element has no specific attributes. You should use CSS properties or techniques such as CSS Grid or CSS Flexbox to align and position The Content Division element elements on the page.

content

<hr>

The Thematic Break (Horizontal Rule) element represents a thematic break between paragraph-level elements (for example, a change of scene in a story, or a shift of topic with a section); historically, this has been presented as a horizontal rule or line. While it may still be displayed as a horizontal rule in visual browsers, this element is now defined in semantic terms, rather than presentational terms, so if you wish to draw a horizontal line, you should do so using appropriate CSS.

This element's attributes include:

  • align
  • color
  • noshade
  • size
  • width

Example:
<p> text </p>

<img>

The Image Embed element embeds an image into the document.

This element's attributes include:

  • alt
  • crossorigin
  • decoding
  • height
  • importance
  • ismap
  • referrerpolicy
  • sizes
  • src
  • srcset
  • width
  • usemap

Example:
<img src="url_or_relative_path" alt="image_description">
